Every acronym that has been mentioned in a lecture
Acronym Stands for... What does it do?
ADSL Asymmetric Digital Subscriber Line Connects home subscribers to the Internet over the old telephone service using PPP protocol
AM Amplitude Modulation As in AM Radio
ARQ Automatic Repeat reQuest Flow control protocol used to manage refusal and acceptance of new device connections in a network (also known as PAR protocol)
ARP Address Resolution Protocol Internet protocol that gives a host an IP address. NO LONGER USED SINCE IT IS MANUAL. DHCP is used instead!
ARPANET Advanced Research Projects Agency Network The predecessor to the Internet.
ATM Asynchronous Transfer Model Involved during the last leg of the journey from the ISP to your home.
BGP Border Gateway Protocol Protocol that manages how packets are routed across the internet through the exchange of routing and reachability information between edge routers.
BOOTP Bootstrap Protocol Internet protocol that gives a host an IP address. NO LONGER USED SINCE IT IS MANUAL. DHCP is used instead!
BSC Base Station Controller Part of the GSM
CDMA Code Division Multiple Access Multiplexing technique that shares the channel by giving specific code to users (used in 3G networks)
CIDR Classless Inter Domain Routing Notation used in classless IPv4 address.
CSMA Carrier Sense Multiple Access
CTS Clear To Send Part of IEEE 802.11 for the MACA protocol for safe communication in wireless LAN's. Solves the Exposed Terminal Problem.
DHCP Dynamic Host Configuration Protocol Internet protocol that automatically assign a local IP address to a host.
DNS Domain Name System Maps IP addresses to domain names
DSL Digital Subscriber Line The 1st technology used to give homes broadband connection. DSL uses pre-existing telephone lines.
ECN Explicit Congestion Notification Field in the IP header to indicate whether there is congestion in the network. If there is congestion, ECN = 11
EGP Exterior Gateway Protocol Group of protocols used for routing algorithms
FDDI Fibre Distributed Data Interface Data transmission using fiber optic cables. Often used to connect lots of routers from a WAN.
FDM Frequency Division Multiplexing Multiplexing technique that shares the channel by placing users on different frequencies
FM Frequency Modulation As in FM Radio
FTP File Transfer Protocol
GEO Geo-stationary Earth Orbit Type of communication satellite (35,000km above ground)
GSM Global System for Mobile communications The global system for mobile communications, main technology used in 2G.
HLDC High Level Data Control Data-link protocol
HLR Home Location Register Part of GSM - database that stores last known location of mobile phones in a country
HTTP Hyper Text Transfer Protocol
ICMP Internet Control Message Protocol Error-reporting protocol used in the Internet - routers use it to generate error messages to the source IP address when network problems prevent delivery of IP packets.
IEEE Institute of Electronic Engineers
IMAP Internet Message Access Protocol Protocol used by the Application layer for mail transfer
IGP Interior Gateway Protocol Group of protocols used for routing algorithms
IHL Internet Header Length IPv4 field to indicate length of IP header
IPP Internet Printing Protocol
IOC Internet Over Cable The 2nd technology used to give homes broadband connection. IOC uses pre-existing TV cable.
ISP Internet Service Provider
IXP Internet eXchange Point
LAN Local Area Network
LEO Low Earth Orbit Type of communication satellite (5,000km above ground)
MACA Multiple Access Collision Avoidance Wireless LAN protocol
MAN Metropolitan Area Network
MANET Mobile Adhoc NETwork
MIME Multipurpose Internet Mail Extensions
MEO Middle Earth Orbit Type of communication satellite (15,000km above ground)
MPLS Multi Protocol Label Switching Protocol used in the Network layer to route packets. Works on connection-orientated services to make virtual circuits
MSC Mobile Switching Centre Part of the GSM - talks to its two databases (VLR and HLR) and is connected to a PSTN.
MTU Maximum Transmission Unit
NAT Network Address Translation Maps private IP address to a public IP address
NAV Network Allocation Vector
NIC Network Interface Card
NNTP Network News Transfer Protocol
NSAP Network Service Access Point Basically an IP address
OSPF Open Shortest Path First Routing protocol, it is a type of IGP and uses an LSR algorithm. It uses a shortest path first algorithm where distances are based on physical distance/delay.
PAN Personal Area Network
PAR Positive Acknowledgement Protocol Same thing as ARQ
PPP Point to Point Protocol Data-link protocol
POP3 Post Office Protocol v3 Protocol used by the Application layer for mail transfer
PSTN Public Switch Telephone Network Part of GSM
RARP Reverse Positive Acknowledgement Protocol Internet protocol that gives a host an IP address. NO LONGER USED SINCE IT IS MANUAL. DHCP is used instead!
RFID Radio Frequency Identification Device
RIP Router Information Protocol Distance vector protocol. It is an older version of OSPF.
RTP Real Time Protocol
RTSP Real Time Streaming Protocol
RTS Request To Send Part of IEEE 802.11 for the MACA protocol for safe communication in wireless LAN's. Solves the Exposed Terminal Problem.
SCTP Streaming Control Transmission Protocol Protocol used to do Downward Multiplexing.
SIM Subscriber Identity Module Part of GSM - What activates your mobile phone and connects it to a network. Invented in 1991.
SONET
SMTP Simple Mail Tranfer Protocol Protocol used by the Application layer for mail transfer
STP Shielded Twisted Pair A type of twisted pair cable
TCP Transfer Control Protocol Reliable connection-orientated protocol. Used to do Upward Multiplexing.
TDM Time Division Multiplexing Multiplexing technique that shares a channel over time
TPDU Transport Protocol Data Unit Alternative name for segment (the data abstraction at the Transport Layer)
TSAP Transport Service Access Point Basically a port in the Internet
UART Universal Asynchronous Receiver Transmitter
UDP User Datagram Protocol Unreliable connection-less protocol. Used during streaming, video/voice calls.
UHF Ultra High Frequency Type of radiowave
USB Universal Serial Bus
UTP Unshielded Twisted Pair Another type of twisted pair cable
VLR Visitor Location Register Part of GSM - database that stores last known location of mobile phones in a network
VSAT Very Small Aperture Terminal A two-way satellite ground station with a dish antenna that is smaller than 4 meters. They bounce signals off larger transmitters
VHF Very High Frequency Type of radiowave
WAN Wide Area Network
