Types of Networks

There are 4 main types of networks:

  • PAN = Personal Area Network
  • LAN = Local Area Network
  • MAN = Metropolitan Area Network
  • WAN = Wide Area Network
Detailed description of each network type
Network type Distance Description Wired examples Wireless examples Main technology
PAN 1m Used in phones, computers, servers USB, SATA, DATA (stuff in your PC) Bluetooth, infrared, microwave Broadcast and point-to-point
LAN 10m - 1km Used in offices, schools, houses Ethernet (100-1000 mbps), coaxial cables Wi-Fi (11-100 mbps) Broadcast
MAN 10km Used to connect devices over a city Fibre optic cables. n/a Point-to-point
WAN 100km - 1000km Used to connect devices across a country or continent Transmission lines (depends on subnet) 3G, 4G, satellite systems Point-to-point
