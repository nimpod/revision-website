Parallel and Serial Transmission

This is the 3rd and final data transmission method.

There are 2 possibililites

  • Parallel
  • Serial
Parallel vs Serial Transmission
Type Description Advantages Disadvantages
Parallel
  • Transmit a group of bits simultaneously.
  • Thick cables -> expensive over long distances.
  • Over long distances there can be synchronisation problems due to bit skew.
  • High bit-rate
  • Bit skew
  • Synchronisation issues
Serial
  • Transmit 1 bit at a time.
  • You would expect this to be slower and less efficient than parallel transmission, but serial transmission is actually faster!
  • Serial transmission is more reliable and cheaper than parallel transmission, especially over long distances.
  • However, the sender must determine the order in which the bits were originally sent, then the receiver must determine in which order to reconstruct a byte. This like the difference between Big Endian (left->right), and Little Endian (right->left)
  • Reliable
  • Cheaper transmission over long distances
  • Sending and receiving can be more complex
