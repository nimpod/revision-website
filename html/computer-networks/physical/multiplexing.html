Multiplexing

If you want to send more than 1 conversation(s) across a single link, multiplexing is your friend.

What do I mean by a 'conversation'? - The communication between 2 devices that carries and transmits some data with it.

Multiplexing allows the carrying of multiple signals on a single medium.

This allows more efficient use of the transmission medium, because more devices can use the link at the same time!

This is the whole purpose behind doing multiplexing - to efficiently use the transmission medium.

There are 3 main multiplexing techniques

  • Frequency Division Multiplexing (FDM)
  • Time Division Multiplexing (TDM)
  • Code Division Multiple Access(CDMA)

Frequency Division Multiplexing (FDM)

Similar to a motorway.

FDM is a multiplexing technique that shares the channel by placing users on different frequencies.

"break down the channel into multiple channels, each with a specific frequency, and we divide the channels among different hosts."

So we are basically 'dividing by the frequency' hence the name FDM.

In other words, we shift each signal to a different carrier frequency.

The total bandwidth available is divided into a series of non-overlapping frequency bands, each of which is used to carry a separate signal. Kind of like a motorway, where you can send multiple cars down one lane, and there can be multiple lanes per motorway.

For FDM to work properly frequency overlap must be avoided. Therefore, the link must have sufficient bandwidth to be able to carry the wide range of frequencies required.

Time Division Multiplexing (TDM)

Similar to Round Robin (oh those were dark times).

Use the entire bandwidth, but only for short time periods.

Instead of dividing the frequencies, we divide the time into separate slots. Then we give those slots to different conversations. (It's sorta like how Round Robin works because they are taking turns in using the link.)

Takes advantage of the fact that achievable bit rate of medium exceeds the required data rate of a digital signal.

Code Division Multiple Access (CDMA)

Shares the channel by giving user specific codes.

Distinguishing between conversations is just a case of differentiating between those codes. You could then tune the network to only listen to conversations with a particular code.

Allows conversations to use the whole bandwidth at any time.

Commonly used in 3G networks.
