Every terms that has been mentioned in a lecture
Term What does it do?
Adaptive Routing algorithms Adaptive algorithms attempt to change routing decisions to adjust to traffic changes or topology changes. (e.g. Distance Routing Vector algorithm, Dijkstra's Algorithm, etc...)
Backpressure If a router has no free buffers, it must ignore newly arriving packets. When a packet is discarded, the sending router times out and retransmits, until the packets get through. This router must then keep the packets longer in its queues, so it becomes itself congested and the problem spreads (called backpressure).
Border router A router than connects two networks together. It sits on the border of a network, holding onto a connection with another network.
Choke Packet Used in the congestion control approach called 'Traffic Throttling'.
If a router thinks the network is too congested, the router will send a choke packet to the original sender.
The choke packet is a way of telling the sender to lower its transmission rate.
This process may be repeated several times, until the router is satisfied with the state of congestion in the network.
Congestion Collapse When too large of a buffer is used, if the time taken for a packet to travel from the beginning -> end of the buffer is > timeout period (which is more likely with a larger buffer), packets will be lost.
Downward Multiplexing Multiple network addresses are available to 1 host application. The host application can spread its data to each network address. (Network connections share a host application). This type of multiplexing is Implemented using the SCTP protocol.
Home agent vs. Foreign agent The Home agent keeps track of the devices that are leaving the network.
The Foreign agent keeps track of the devices coming into its network.
Internetworking This is when you combine different networks that use different technologies. They are connected using border routers.
Jitter When an unreliable network is supplying a real-time service, and packets are streamed by their arrival time, the video stream will jitter
Jitter means the video and audio could be jumpy, stopping and starting, and displaying packets out of order.
We use buffers to reduce jitter in real-time transport protocols.
Passband Modulation ...
Piggy backing Piggy backing is when you combine the data and acknowledgement into one dataframe. The ACK literally gets 'piggy-backed' on the data.
Means there are less dataframes going backwards and forwards, resulting in a better channel utilisation. Also more optimal computation.
It is used to improve the ARQ protocol.
Time to live (TTL) field One of the fields in the IPv4 header. It determines the amount of time that a packet should remain in a network for.
TTL is a number that is decremented by 1 each time it arrives at a new router.
When TTL = 0, the packet is discarded forever.
The TTL field reduces congestion because packets will not roam around the network forever.
Upward Multiplexing If only one network address is available on a host, all transport layer connections have to use it (Transport connections share a network address).
